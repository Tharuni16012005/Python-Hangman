# 🎮 Hangman Game in Python

## 📌 Project Description

This is a simple **Hangman Game** developed using Python. The game randomly selects a word, and the player must guess it one letter at a time within a limited number of attempts.

It is a beginner-friendly project designed to demonstrate core Python concepts such as loops, conditionals, functions, and string handling.

---

## 🚀 Features

* Random word selection
* Hint system for each word
* Limited attempts (lives)
* Input validation
* Win/Loss conditions
* User-friendly console interface

---

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run the Project

1. Install Python (if not already installed)
2. Download or clone this repository:

   ```
   git clone https://github.com/your-username/hangman-game.git
   ```
3. Navigate to the project folder:

   ```
   cd hangman-game
   ```
4. Run the Python file:

   ```
   python hangman.py
   ```

---

## 🎯 How to Play

* You will be given a hint about the word.
* Guess one letter at a time.
* Each wrong guess reduces your attempts.
* Guess the complete word before your attempts run out to win!
